# ZbarScanLibrary
Zar scan open source

ZbarScanLibrary是之前的ZBar解析二维码项目的重构和优化


>* 使用了最新版本的libiconv库进行编译，版本号：1.14；下载地址： http://www.gnu.org/software/libiconv/
>* 重新修改ZBar对中文的支持，并且可以解析GBK，UTF-8格式生成的二维码图片
>* 使用最新的ndk-r10d进行ndk-build，支持64位cpu
